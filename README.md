# EWRC

Easy Web Rest Client

### Get

Install: 

```
go get -u github.com/z0rr0/ewrc
```

Build:

```
go install github.com/z0rr0/ewrc
```

Run example:

```
./ewrc -h localhost -p 1001 -d db.sqlite3
```
